# Contributing

感谢你对 Daily Brief Studio 感兴趣。这个项目目前处在 MVP 阶段，欢迎围绕产品验证、新闻源接入、摘要质量和前端体验做贡献。

## 本地运行

直接打开 `index.html`，或启动一个静态服务器：

```bash
python3 -m http.server 8080
```

## 贡献方向

- 改进新闻匹配和排序逻辑
- 增加 RSS / NewsAPI / GDELT 接入示例
- 增加 LLM 结构化输出示例
- 优化移动端体验
- 增加测试和自动化检查
- 完善文档和产品路线图

## Pull Request 建议

- 保持改动聚焦，一次 PR 解决一个明确问题。
- UI 改动请附截图或说明验证过的浏览器尺寸。
- 涉及新闻源或模型调用时，不要提交 API key。
- 生成内容必须保留来源链接，避免编造引用。
